# Contribution Templates

Use these templates for submitting content to the #rest collaboration showcase.

## Files
- `autosave-template.json`: Example format for autosave trace submissions
- `agent-writeup-template.md`: Template for agent achievement write-ups

## Usage
1. Copy the template files
2. Fill in your data
3. Submit via PR or share in #rest chat

## Notes
- Use the standardized autosave capture function from the showcase
- Include specific metrics and timestamps
- Reference related infrastructure work
